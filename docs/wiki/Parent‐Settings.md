# Parent Settings

## Customizable Options

- Child Age
- Mode: Strict / Balanced / Relaxed
- Block Adult
- Block Gambling
- Block Phishing
- Warn Suspicious

---

## Whitelist

Domains always allowed.

Example:
- google.com
- wikipedia.org

---

## Blacklist

Domains always blocked.

Example:
- badsite.com
- xxx-site.com
