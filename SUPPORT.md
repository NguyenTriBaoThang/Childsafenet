# Support

Thank you for using ChildSafeNet ❤️

If you need help, please follow the steps below.

---

## 📌 Documentation

- Check the README.md first.
- Review the installation steps for Backend, Frontend, and AI Service.
- Make sure environment variables are configured correctly.

---

## 🐛 Reporting Bugs

Please use the **Bug Report** issue template.

Include:

- Steps to reproduce
- Expected behavior
- Logs (if available)
- Screenshots

---

## 💡 Feature Requests

Use the **Feature Request** template.

---

## 🔐 Security Issues

Do NOT open public issues.

Instead, email:
security@yourdomain.com

Include:

- Description
- Reproduction steps
- Impact assessment

---

## 🤝 Community Support

You may:

- Open a GitHub Discussion
- Open a draft issue for technical questions

---

Thank you for contributing to a safer internet for children.
